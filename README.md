# Wizard Battleship
Une bataille navale pas comme les autres
## Présentation
**Wizard Battleship** est une version améliorée du jeu **Bataille Navale**.  
Le projet ajoute :
- une **intelligence artificielle (IA)** avec plusieurs niveaux de difficulté,  
- des **cartes spéciales** permettant des effets stratégiques et amusants,  
- et une interface console claire et interactive.

Ce projet a été réalisé dans le cadre du cours **Algorithmique et Programmation (L2 MIASHS)**.

## Fonctionnalités principales

### Jeu de base
- Grille   
- Placement automatique ou manuel des bateaux  
- Tirs successifs avec résultat (“Touché”, “Coulé”, “Raté”)  
- Fin de partie lorsqu’un joueur perd tous ses bateaux  

### Intelligence Artificielle
Le joueur peut choisir le niveau de l’IA :    
1. **Moyen** → tirs autour des cases touchées  
2. **Difficile** → stratégie avancée (probabilités ou pattern checkerboard)
